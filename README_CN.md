# OpenArt-mainline

**主线版本 · 仅限非商业用途**

OpenArt-mainline 是 OpenArt 许可证家族的非商业主线版本，专为对抗各类可能导致艺术作品缺损的政策、平台规则或技术处理而设计。它构建了一个**由艺术家自主维护的信任网络（Trust Network）**，确保未经修正的艺术作品能够在网络中自由流动，同时完整保留艺术家的署名权。

## 最高目标

**让艺术回归完整，让信任取代强制。**

- **对抗缺损**：任何形式的有损压缩、色彩配置篡改、自动水印、格式强制转换等行为，均被视为对作品完整性的侵害。
- **技术自立**：通过加密压缩包、宽松平台、色彩配置校验等手段，使艺术品的传播不依赖于任何可能损害作品的中间方。
- **信任优先于规则**：在信任网络内部，艺术家的共同承诺优先于任何相冲突的法律、政策或平台条款。
- **不依赖外部认可**：许可证的效力完全来自成员的自愿遵守与相互监督，不寻求任何外部法律或行政支持。

## 核心方法

### 1. 信任网络

- 任何使用本许可证发布作品的艺术家**自动成为信任网络成员**。
- 成员之间**互为授权**，无需逐份签署协议即可自由交换无修正作品。
- **信任传递**：从网络中获得的作品，其权利范围一律以本许可证为准。
- **即便原作者退出网络，已分发的作品仍可继续由他人再分发，无需额外同意。**

### 2. 无修正的严格定义

一个作品被视为“未经修正”必须同时满足：

- **核心内容数据严格相等**：像素矩阵、采样序列、帧序列、顶点数据等与原始状态**逐点完全相同**（允许无损压缩，解压后必须一致）。
- **色彩配置完全一致**：ICC 配置文件、色彩空间定义等**不得变更、移除或替换**（若无嵌入，则视为默认 sRGB，且不得添加）。
- **其他元数据（EXIF、XMP 等）可以任意修改**，不影响无修正状态。

### 3. 分发渠道限制

- **优先使用“宽松平台”**：不对作品进行有损压缩、不添加水印、不限制原始文件下载。
- 无法使用宽松平台时，必须采用 **AES‑256 加密的无损压缩包**，密码通过安全信道单独传递。
- **禁止设置利益门槛**：不得要求接收方执行任何为原作者带来利益的操作（打赏、关注、点赞、支付等）。允许接收方安装第三方应用或注册账号，只要原作者从中不获利。

### 4. 外部强制的宽容处理

如果艺术家**因法律或平台政策的强制性要求**（非自愿）不得不对作品进行缺损处理：

- 其信任网络成员资格**不会自动中止**。
- 但必须明确标注修正版“非 OpenArt-mainline 无修正版本”，且不得将该版本作为无修正作品分发。
- 应尽最大努力通过其他渠道提供无修正版本。

### 5. 非商业使用

- 本许可证**严格禁止任何商业用途**（销售、广告、嵌入商业产品、营利性服务等）。
- 个人学习、网络内部非营利分享、不收费的非商业展览不受影响。
- 需要商业授权请等待 **OpenArt-commerce** 版本。

## 大概规则一览

| 规则领域 | 具体要求 |
|---------|----------|
| **再分发权利** | 任何成员可无需原作者同意再次分发无修正作品；原作者退出网络不影响已分发作品。 |
| **作品完整性** | 核心数据 + 色彩配置必须与原始状态完全相同；有损压缩、色彩配置变更即构成缺损。 |
| **分发渠道** | 优先宽松平台；备选加密无损压缩包；严禁设置打赏、关注、点赞等利益门槛。 |
| **署名** | 必须标注作者、作品名、许可证名；可使用文件名或附随文本，不强制依赖元数据。 |
| **成员资格中止** | 自愿进行有损处理或设置利益门槛 → 自动中止；外部强制 → 不中止但须标注。 |
| **法律冲突** | 信任网络内部优先遵守本许可证；因遵守本许可证而违法当地法律的责任由行为人自负。 |
| **商业限制** | 完全禁止商业用途；商业需求请使用商业版。 |

## 如何使用

### 作为发布者

1. 在您的作品（图像、源文件等）中声明：  
   `本作品采用 OpenArt-mainline 许可证进行授权。`
2. 同时提供**无修正原作**（核心数据+色彩配置完整）和**公开预览版**（可有损/低分辨率）。
3. 将作品上传到**宽松平台**，或制作加密压缩包并公开密码（无利益门槛）。
4. 您自动成为信任网络成员，可以自由获取和分享其他成员的无修正作品。

### 作为使用者

- 您可以自由下载、使用、再分发无修正作品，但**必须遵守相同的完整性保护和署名义务**。
- **不得**将作品上传到有损压缩平台、添加水印、修改色彩配置，也不得设置利益门槛。
- 违反后所有授权自动终止，须销毁无修正版本。
