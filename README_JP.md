# OpenArt-mainline

**メインライン版 · 非商業利用限定**

OpenArt-mainline は OpenArt ライセンスファミリーの非商業メインライン版です。芸術作品に非自発的な欠損をもたらす可能性のある政策、プラットフォームルール、技術的処理に対抗し、**アーティストが自ら維持するトラストネットワーク**を構築します。このネットワークにおいて、無修正の芸術作品は自由に流通し、アーティストの氏名表示権は完全に保持されます。

## 最高目標

**芸術を完全なものにし、強制を信頼に置き換える。**

- **欠損への抵抗** – 非可逆圧縮、色設定の改変、自動透かし、強制フォーマット変換などは、すべて作品の完全性に対する侵害とみなします。
- **技術的自立** – 暗号化アーカイブ、寛容的プラットフォーム、色設定検証などの技術的手段により、作品の伝搬を損傷の恐れのある中間者に依存させません。
- **ルールより信頼** – トラストネットワーク内部では、アーティストの集合的コミットメントが、矛盾する法律・政策・プラットフォーム規約よりも優先されます。
- **外部承認に依存しない** – 本ライセンスの効力は、メンバーの自発的遵守と相互監督のみに由来し、外部の法的・行政的支援を求めません。

## 核となる方法

### 1. トラストネットワーク

- 本ライセンスの下で作品を公開する**すべての個人・団体は自動的にトラストネットワークメンバー**となります。
- メンバーは**相互に許諾**し合います。無修正作品を交換するために個別の契約は不要です。
- **信頼の移転** – ネットワークから入手した作品の権利範囲は、常に本ライセンスのみに基づきます。
- **原作者がネットワークを脱退しても、既に配布された作品は追加の同意なしに他のメンバーが再配布できます。**

### 2. 「無修正」の厳格な定義

作品が「無修正」と見なされるには、以下の**両方**を満たす必要があります。

- **コアコンテンツデータが厳密に同一** – ピクセル行列、サンプル系列、フレーム系列、頂点データなどが、オリジナルの状態と**ビット単位で完全に同一**（無損失圧縮は許可されるが、解凍後は完全に同一でなければならない）。
- **色設定が完全に同一** – ICCプロファイル、色空間定義などは**変更、削除、置換されてはならない**（オリジナルに埋め込まれた色設定がない場合は、デフォルト色空間（通常 sRGB）を使用しているとみなし、追加もできない）。
- **その他のメタデータ（EXIF、XMPなど）は自由に変更可能** – 無修正のステータスに影響しません。

### 3. 配布チャネルの制限

- **「寛容的プラットフォーム」を優先** – 非可逆圧縮を適用せず、透かしを追加せず、オリジナルファイルのダウンロードを制限しないプラットフォーム。
- 寛容的プラットフォームが利用できない場合は、**AES-256暗号化・無損失圧縮アーカイブ**で配布し、パスワードは安全なチャネルで別途送付。
- **利益障壁の禁止** – 原作者に直接的・間接的に利益をもたらす行為（投げ銭、フォロー、いいね、支払いなど）を、作品入手の条件として要求してはならない。サードパーティ製アプリのインストールやアカウント登録は、原作者が利益を得ない限り許可されます。

### 4. 外部強制欠損への寛容な扱い

アーティストが**法律またはプラットフォームポリシーの強制的要件**（非自発的）によりやむを得ず作品に欠損を生じさせた場合：

- そのトラストネットワークメンバー資格は**自動的には停止されません**。
- ただし、当該欠損版に「OpenArt-mainline 無修正版ではない」旨を明示し、**無修正作品としてネットワーク内で配布してはなりません**。
- 他のチャネル（暗号化アーカイブ、代替プラットフォームなど）を通じて無修正版を提供する合理的な最大限の努力をしなければなりません。

### 5. 非商業利用

- 本ライセンスは**商業利用を厳格に禁止**します（販売、広告、商業製品への組込み、営利サービスなど）。
- 個人学習、ネットワーク内での非営利共有、入場料無料の非商業展示は許可されます。
- 商業的需要がある場合は **OpenArt-commerce** 版を待ってください。

## 主要ルールの概要

| 分野 | 具体的要件 |
|------|-------------|
| **再配布権** | いかなるメンバーも、原作者の追加同意なしに無修正作品を再配布できる。原作者の脱退は既配布作品に影響しない。 |
| **作品の完全性** | コアデータ＋色設定がオリジナルと厳密に同一でなければならない。非可逆圧縮や色設定の変更は欠損とみなす。 |
| **配布チャネル** | 寛容的プラットフォームを優先。代替は暗号化無損失アーカイブ。投げ銭・フォロー・いいね等の利益障壁を禁止。 |
| **氏名表示** | 作者名、作品名、ライセンス名を明示。ファイル名や付随テキストでも可。メタデータへの依存は強制しない。 |
| **メンバー資格停止** | 自発的欠損処理や利益障壁の設定 → 自動停止。外部強制欠損 → 停止せず但し明示義務。 |
| **法律との衝突** | トラストネットワーク内では本ライセンスが優先。本ライセンス遵守の結果現地法に違反した責任は行為者個人が負う。 |
| **商業制限** | 商業利用は完全禁止。商業的需要は商業版を使用。 |

## 使い方

### 公開者として

1. 作品（画像、ソースファイル等）に以下の宣言を記載：  
   `この作品は OpenArt-mainline ライセンスの下で提供されます。`
2. **無修正オリジナル**（コアデータ＋色設定完全）と**公開プレビュー版**（低解像度/非可逆可）の両方を提供。
3. **寛容的プラットフォーム**にアップロードするか、暗号化アーカイブを作成してパスワードを公開（利益障壁禁止）。
4. 自動的にトラストネットワークメンバーとなり、他のメンバーの無修正作品を自由に入手・共有できる。

### 利用者として

- 無修正作品のダウンロード、使用、再配布は自由だが、**同一の完全性保護と氏名表示義務を遵守しなければならない**。
- 非可逆圧縮プラットフォームへのアップロード、透かし追加、色設定変更、利益障壁設定をしてはならない。
- 違反すると全ての権利が自動終了し、入手した無修正版を廃棄しなければならない。
