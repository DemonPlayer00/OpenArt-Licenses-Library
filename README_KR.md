# OpenArt-mainline

**메인라인 버전 · 비상업적 용도 전용**

OpenArt-mainline은 OpenArt 라이선스 패밀리의 비상업적 메인라인 버전입니다. 예술 작품에 비자발적 훼손을 초래할 수 있는 정책, 플랫폼 규칙, 기술적 처리를 방어하고, **아티스트가 자체 유지하는 트러스트 네트워크**를 구축하여 무수정 예술 작품이 자유롭게 유통되고 아티스트의 성명 표시권이 완전히 보존되도록 합니다.

## 최고 목표

**예술을 온전하게, 강제를 신뢰로 대체하자.**

- **훼손 저항** – 비가역 압축, 색상 구성 변경, 자동 워터마크, 강제 형식 변환 등은 모두 작품 완전성에 대한 침해로 간주합니다.
- **기술적 자립** – 암호화 아카이브, 관용적 플랫폼, 색상 구성 검증 등의 기술적 수단을 통해 작품 유통을 훼손 가능성이 있는 중개자에 의존하지 않습니다.
- **규칙보다 신뢰** – 트러스트 네트워크 내에서는 아티스트의 집단적 약속이 상충하는 법률, 정책, 플랫폼 약관보다 우선합니다.
- **외부 승인 불필요** – 본 라이선스의 효력은 오직 구성원의 자발적 준수와 상호 감독에서 비롯되며, 외부 법률이나 행정 지원을 구하지 않습니다.

## 핵심 방법

### 1. 트러스트 네트워크

- 본 라이선스 하에서 작품을 공개하는 **모든 개인 또는 단체는 자동으로 트러스트 네트워크 구성원**이 됩니다.
- 구성원은 **상호 허락**합니다. 무수정 작품을 교환하기 위해 개별 계약이 필요하지 않습니다.
- **신뢰 이전** – 네트워크에서 획득한 작품의 권리 범위는 항상 본 라이선스에만 따릅니다.
- **원저작자가 네트워크를 탈퇴하더라도 이미 배포된 작품은 다른 구성원이 추가 동의 없이 계속 재배포할 수 있습니다.**

### 2. “무수정”의 엄격한 정의

작품이 “무수정”으로 간주되려면 다음 **두 가지 조건을 모두** 만족해야 합니다.

- **코어 콘텐츠 데이터가 엄격히 동일** – 픽셀 행렬, 샘플 시퀀스, 프레임 시퀀스, 정점 데이터 등이 원본 상태와 **비트 단위로 완전히 동일** (무손실 압축은 허용되나 압축 해제 후 원본과 완전히 동일해야 함).
- **색상 구성이 완전히 동일** – ICC 프로파일, 색상 공간 정의 등은 **변경, 제거 또는 대체되어서는 안 됨** (원본에 내장된 색상 구성이 없는 경우 기본 색상 공간(보통 sRGB)을 사용하는 것으로 간주하며, 추가할 수도 없음).
- **기타 메타데이터(EXIF, XMP 등)는 자유롭게 변경 가능** – 무수정 상태에 영향을 미치지 않습니다.

### 3. 배포 채널 제한

- **“관용적 플랫폼” 우선 사용** – 비가역 압축을 적용하지 않고, 워터마크를 추가하지 않으며, 원본 파일 다운로드를 제한하지 않는 플랫폼.
- 관용적 플랫폼을 사용할 수 없는 경우, **AES-256 암호화 무손실 압축 아카이브**로 배포하고 비밀번호는 안전한 채널로 별도 전송.
- **이익 장벽 금지** – 원저작자에게 직접적 또는 간접적 이익을 가져오는 행위(후원, 팔로우, 좋아요, 결제 등)를 작품 획득 조건으로 요구할 수 없습니다. 제3자 앱 설치나 계정 등록은 원저작자가 이익을 얻지 않는 한 허용됩니다.

### 4. 외부 강제 훼손에 대한 관용

아티스트가 **법률 또는 플랫폼 정책의 강제적 요구**(비자발적)로 인해 부득이하게 작품에 훼손을 가한 경우:

- 트러스트 네트워크 구성원 자격이 **자동으로 정지되지 않습니다**.
- 단, 해당 훼손 버전에 “OpenArt-mainline 무수정 버전이 아님”을 명시하고, **무수정 작품으로 네트워크 내에서 배포해서는 안 됩니다**.
- 다른 채널(암호화 아카이브, 대체 플랫폼 등)을 통해 무수정 버전을 제공하기 위해 합리적인 최대 노력을 다해야 합니다.

### 5. 비상업적 사용

- 본 라이선스는 **상업적 사용을 엄격히 금지**합니다(판매, 광고, 상업 제품에 포함, 유료 서비스 등).
- 개인 학습, 네트워크 내 비영리 공유, 입장료 없는 비상업적 전시는 허용됩니다.
- 상업적 필요가 있는 경우 **OpenArt-commerce** 버전을 기다리십시오.

## 주요 규칙 요약

| 영역 | 구체적 요구사항 |
|------|----------------|
| **재배포 권리** | 모든 구성원은 원저작자의 추가 동의 없이 무수정 작품을 재배포할 수 있음. 원저작자 탈퇴는 이미 배포된 작품에 영향을 미치지 않음. |
| **작품 완전성** | 코어 데이터 + 색상 구성이 원본과 엄격히 동일해야 함. 비가역 압축이나 색상 구성 변경은 훼손으로 간주. |
| **배포 채널** | 관용적 플랫폼 우선. 대안은 암호화 무손실 아카이브. 후원, 팔로우, 좋아요 등의 이익 장벽 금지. |
| **성명 표시** | 저작자명, 작품명, 라이선스명을 명시. 파일명이나 부속 텍스트 사용 가능. 메타데이터 의무 아님. |
| **구성원 자격 정지** | 자발적 훼손 처리나 이익 장벽 설정 → 자동 정지. 외부 강제 훼손 → 정지 없으나 표시 의무. |
| **법률 충돌** | 트러스트 네트워크 내에서는 본 라이선스 우선. 본 라이선스 준수로 인한 현지 법률 위반 책임은 행위자 개인이 부담. |
| **상업적 제한** | 상업적 사용 완전 금지. 상업적 필요 시 상업용 버전 사용. |

## 사용 방법

### 공개자(게시자)로서

1. 작품(이미지, 소스 파일 등)에 다음 선언을 포함:  
   `이 작품은 OpenArt-mainline 라이선스로 제공됩니다.`
2. **무수정 오리지널**(코어 데이터+색상 구성 완전)과 **공개 미리보기 버전**(저해상도/비가역 허용)을 모두 제공.
3. **관용적 플랫폼**에 업로드하거나, 암호화 아카이브를 생성하여 비밀번호를 공개(이익 장벽 금지).
4. 자동으로 트러스트 네트워크 구성원이 되며, 다른 구성원의 무수정 작품을 자유롭게 획득하고 공유할 수 있음.

### 이용자로서

- 무수정 작품을 다운로드, 사용, 재배포할 수 있으나 **동일한 완전성 보호 및 성명 표시 의무를 준수해야 함**.
- 비가역 압축 플랫폼에 업로드, 워터마크 추가, 색상 구성 변경, 이익 장벽 설정을 해서는 안 됨.
- 위반 시 모든 권리가 자동 종료되며, 획득한 무수정 버전을 폐기해야 함.
